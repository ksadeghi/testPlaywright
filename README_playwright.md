# testPlaywright

A test project for Playwright automation testing.

## Overview

This project demonstrates Playwright testing capabilities with examples and test cases.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Install Playwright browsers:
```bash
npx playwright install
```

## Running Tests

```bash
# Run all tests
npm test

# Run tests in headed mode
npm run test:headed

# Run tests with UI mode
npm run test:ui
```

## Project Structure

- `tests/` - Test files
- `playwright.config.js` - Playwright configuration
- `package.json` - Project dependencies and scripts

## Getting Started

This project includes sample tests to demonstrate Playwright's capabilities for web automation and testing.
